WISEBUDGET - aka Wisebuddy

#Features:#
- Displays all transactions in a table.
- Form for adding new transactions.
- Flash messages for user feedback.
- Expense breakdown and balance trend charts.
- Button to open the category color edit modal.
- Responsive, modern UI with usage of Bootstrap and FontAwesome.


This application offers a robust, modular, and extensible finance tracking system. Each file has a clear responsibility, from database access to user interface. 
The use of Flask's routing, context-managed DB operations, and AJAX-driven UI updates creates a modern and user-friendly experience. 
The code is ready to be extended with further analytics, authentication, or multi-user capabilities.
